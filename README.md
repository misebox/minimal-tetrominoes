# minimal-tetrominoes

tetrominoes written in nim

```
    Tetrominoes


    |          |
    |          |
    |          |
    |   ####   |
    |          |
    |          |
    |          |
    |          |
    |          |
    |          |
    |          |
    |          |
    |          |
    |          |
    |          |
    |  ##      |
    |  ####    |
    |  ####    |
    | ##### #  |
    | ### #####|
    |==========|
    SCORE: 450

    ESC: QUIT
    ← , ↓ , → : H, J, L
    DROP: SPACE
    ROTATE L, R: SHIFT+K, K
    PAUSE: P
    BACK TO GAME: ENTER
```

## Requirements
```
apt install termbox-dev
```

## Building
```
git clone git@github.com:misebox/minimal-tetrominoes.git
cd minimal-tetrominoes
nimble build
./minimal-tetrominoes
```
