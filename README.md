# minimal-tetrominoes

tetrominoes written in nim

```
Tetrominoes

|          |
|          |
|          |
|          |
|          |
|      #   |
|      ##  |
|       #  |
|          |
|          |
|          |
|          |
|          |
|          |
|          |
|  ##      |
|  ####    |
|  ####    |
| ##### #  |
| ### #####|
|==========|
score: 0
ESC:quit, lhj:→ ← ↓, Kk:rotate LR, Space: drop
```

## Requirements
```
apt install termbox-dev
```

## Building
```
git clone git@github.com:misebox/minimal-tetrominoes.git
cd minimal-tetrominoes
nimble build
./minimal-tetrominoes
```
